# 01. 하드웨어 준비 및 부품 설명

## 키트 구성품
- 아두이노 UNO (또는 호환 보드)
- MPU6050 자이로/가속도 센서
- L298N 모터 드라이버
- DC 모터 2개
- 바퀴 2개
- 배터리 박스 및 케이블
- 점퍼선, 나사, 프레임 등
- 기타: 블루투스 모듈(옵션), 스마트폰 APP

## 준비물 체크리스트
- 모든 부품이 정상적으로 동봉되어 있는지 확인
- 아두이노 보드와 USB 케이블 준비
- 드라이버, 니퍼 등 간단한 공구

## 조립 개요
1. 프레임에 모터와 바퀴를 장착
2. 모터 드라이버(L298N) 연결
3. MPU6050 센서 고정 및 배선
4. 아두이노와 각 부품 연결
5. 배터리 박스 연결 및 전원 확인

## 실습 목표
- 모든 부품을 올바르게 조립하고, 회로 연결 준비
- 각 부품의 역할과 연결 방법 이해
