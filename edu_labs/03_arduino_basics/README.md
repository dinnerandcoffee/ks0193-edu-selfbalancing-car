# 03. 아두이노 IDE 설치 및 기본 예제

## 목표
- 아두이노 개발 환경(IDE) 설치
- 기본 예제 코드 업로드 및 동작 확인

---

## 아두이노 IDE 설치

### 1. 공식 사이트에서 다운로드
- [Arduino IDE 다운로드](https://www.arduino.cc/en/software)
- Windows, macOS, Linux 모두 지원

### 2. 설치 및 실행
- 설치 후 Arduino IDE 실행
- USB로 아두이노 보드 연결

### 3. 보드/포트 설정
- [도구] → [보드] → "Arduino UNO" 선택
- [도구] → [포트] → 연결된 포트 선택

---

## 기본 예제 업로드

### 1. Blink 예제
- [파일] → [예제] → [01.Basics] → [Blink]
- 업로드 버튼 클릭
- 아두이노의 LED(13번 핀)가 1초 간격으로 깜빡이면 성공

### 2. 시리얼 모니터 확인
- [도구] → [시리얼 모니터] 클릭
- 9600 baudrate 설정
- "Hello, Arduino!" 등 메시지 출력 확인

---

## 실습 체크리스트
- IDE 설치 및 실행 성공
- 보드/포트 인식 확인
- 예제 코드 업로드 및 동작 확인

---

## 문제 해결
- 보드 인식 오류: USB 케이블/포트 변경, 드라이버 재설치
- 업로드 실패: 포트/보드 설정 확인, 아두이노 리셋
